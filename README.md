#  OmikujiApp


## 概要
切符をスキャンしシェイクをすることでおみくじができる「切符で開運おみくじ」

リリース済みのおみくじ機能に加えてGPSチェックイン機能を追加。各駅に表示しているピンの中でチェックインを行うとスタンプを獲得することが出来る。

また、チェックインしたい地点の通知をオンにすることで設定した範囲に入るとプッシュ通知で知らせることが出来る。
